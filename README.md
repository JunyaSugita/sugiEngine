# _Darkness (SugiEngine)_

## 概要
このゲームはたくさんの呪文を使いながら敵を倒すゲームです。

## 攻撃手段

### 打撃
    通常攻撃はためる必要なく使用出来ますがダメージが低く敵に近づかないといけないため
    使いどころを見極めましょう。
![title](https://github.com/JunyaSugita/sugiEngine/assets/94422859/76264822-c2b2-4273-b654-c3cb6d936313)

### 呪文
- ファイアボール

- マジックミサイル

- アイスボルト

- チェインライトニング

- エンチャントファイア
